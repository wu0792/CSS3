总结CSS优先级：

  • 如果有!important样式，5级，优先级最高，应用该样式；
  • 如果有内联样式，4级，即元素内的style定义，应用该样式；
  • 如果有ID样式，3级，应用该样式；
  • 如果有伪类样式，2级，应用该样式；
  • 如果有类型样式，1级，应用该样式；
  • 如果出现组合，上面的各个级别的样式会进行权重叠加，不同浏览器可能不同，可以定位10，
    即 5级样式数目*(10^5) + 4级样式数目*(10^4) + 3级样式数目*(10^3) + 2级样式数目*(10^2) + 1级样式数目*(10^1)
